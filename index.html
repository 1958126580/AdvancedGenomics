üìñ Overview

AdvancedGenomics.jl is a comprehensive, high-performance Julia package that bridges classical quantitative genetics with modern machine learning and deep learning approaches. Designed for researchers, breeders, and bioinformaticians, it provides a unified framework for genome-wide association studies (GWAS), genomic selection (GS), population genetics, and advanced statistical modeling.

The package leverages Julia's exceptional performance characteristics, offering multi-threaded CPU implementations and seamless GPU acceleration for computationally intensive tasks. With over 100+ exported functions spanning statistical genetics, machine learning, deep learning, and visualization, AdvancedGenomics.jl represents a complete toolkit for modern genomic analysis.

üì¶ Installation

‚ö†Ô∏è Important: This package is not yet registered in the Julia General Registry. Install directly from GitHub.

Open the Julia REPL and run:

using Pkg
Pkg.add(url="https://github.com/1958126580/AdvancedGenomics")

For GPU acceleration, also install CUDA.jl:

Pkg.add("CUDA")

Quick Start Example

using AdvancedGenomics

# Load data
genotypes = read_genotypes("data.vcf")
phenotypes = read_phenotypes("traits.csv")

# Run GWAS
results = run_gwas(genotypes, phenotypes)

# Visualize
manhattan_plot(results)
