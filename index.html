üìñ Overview

AdvancedGenomics.jl is a comprehensive, high-performance Julia package that bridges classical quantitative genetics with modern machine learning and deep learning approaches. Designed for researchers, breeders, and bioinformaticians, it provides a unified framework for genome-wide association studies (GWAS), genomic selection (GS), population genetics, and advanced statistical modeling.

The package leverages Julia's exceptional performance characteristics, offering multi-threaded CPU implementations and seamless GPU acceleration for computationally intensive tasks. With over 100+ exported functions spanning statistical genetics, machine learning, deep learning, and visualization, AdvancedGenomics.jl represents a complete toolkit for modern genomic analysis.

üíª Usage Examples

Example 1: GWAS Analysis

using AdvancedGenomics

# Load data
genotypes = read_genotypes("data.vcf")
phenotypes = read_phenotypes("traits.csv")

# Quality control
genotypes_qc = filter_maf(genotypes, 0.05)

# Build genomic relationship matrix
K = build_grm(genotypes_qc)

# Run GWAS with FarmCPU
results = run_farmcpu(phenotypes.trait1, genotypes_qc, K)

# Visualize
manhattan_plot(results, output="manhattan.png")
qq_plot(results, output="qq.png")

Example 2: Genomic Selection

# Build kernel
K = build_grm(genotypes)

# Bayesian Ridge Regression
model = run_bayesA(phenotypes.yield, K, 
                   niter=50000, burnin=10000)

# Predict breeding values
ebv = predict(model, new_genotypes)

# Cross-validation
cv_results = cross_validation(phenotypes.yield, K, 
                               nfolds=5, method="BayesB")

Example 3: Deep Learning

using Lux

# Define Genomic Transformer
model = GenomicTransformer(
    vocab_size = 4,        # A, C, G, T
    embed_dim = 128,
    num_heads = 8,
    num_layers = 6,
    dropout = 0.1
)

# Train model
train_transformer!(model, genotypes, phenotypes,
                   epochs=100, batch_size=32)

# Explainable AI
saliency = saliency_map(model, test_sequence)
plot_saliency(saliency, output="importance.png")

Example 4: Multi-trait Analysis

# Multi-trait LMM
traits = [:yield, :protein, :disease_resistance]
results = run_multitrait_lmm(phenotypes, traits, K)

# Genetic correlations
print(results.genetic_correlations)
